# AI Assistants for Leaders - Workshop Material

Workshop site for Husqvarna Group leadership training on using AI assistants for everyday management tasks.

**Live site:** https://paulinamodlitba.github.io/husqvarna-ai-workshop/

## About

This workshop explores four practical use cases for AI assistants in leadership roles:

1. **Competitive Analysis** - Monitor competitors, analyze market trends, prepare briefings
2. **Difficult Conversations** - Prepare for performance discussions, practice tough feedback
3. **Data Analysis** - Synthesize team reports, identify patterns, prioritize attention
4. **Workshop Preparation** - Design agendas, create exercises, plan facilitation

## What's Included

The downloadable ZIP contains fictional but realistic materials:

| File | Description |
|------|-------------|
| `Business_Context.txt` | Company overview, strategy, competitive landscape |
| `CLAUDE.md` | AI assistant instructions (for Claude Projects) |
| `data/reports/week-06/` | Weekly reports from 4 managers |
| `competitive_intelligence/` | Market update with competitor analysis |
| `templates/` | Frameworks for conversations, workshops, competitor briefs |

## How to Use

1. Download the ZIP file from the website
2. Choose your AI platform (Claude, ChatGPT, or Copilot)
3. Follow the setup instructions on the site
4. Try the example prompts to explore each use case

## Platforms Supported

- **Claude Projects** (claude.ai) - Recommended
- **ChatGPT Projects** (chat.openai.com)
- **Microsoft Copilot** (copilot.microsoft.com)

## Note

All data is **fictional** and created for educational purposes. Never upload sensitive real-world data to AI tools.
