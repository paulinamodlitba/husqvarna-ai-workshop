üõ†Ô∏è Setup Instructions

1

Go to Claude

Visit claude.ai and sign in (or create an account). You need Claude Pro for Projects.

2

Create a New Project

Click "Projects" in the sidebar, then "Create Project". Name it something like "Leadership AI Assistant".

3

Add Custom Instructions

Open CLAUDE.md from the download and copy its contents into the "Custom Instructions" field.

4

Upload Files

In the Project Knowledge section, upload: Business_Context.txt, all weekly reports, and the market intelligence file.

5

Start a Conversation

Click "New Chat" in your project and try one of the example prompts below!

1

Go to ChatGPT

Visit chat.openai.com and sign in. You need ChatGPT Plus for file uploads.

2

Create a New Project

In the sidebar, find "Projects" and create a new one. Name it "Leadership Assistant" or similar.

3

Add Instructions

Click the project settings and paste the contents of CLAUDE.md into the instructions field (it works for any AI).

4

Upload Files

Use the "Add files" option to upload the business context, reports, and intelligence files.

5

Start Chatting

Open a new conversation in your project and try the example prompts below.

1

Open Microsoft Copilot

Go to copilot.microsoft.com or use Copilot in Edge/Microsoft 365.

2

Start a Conversation

Copilot doesn't have persistent projects, so you'll work within a single conversation.

3

Upload Context First

Use the attachment button to upload Business_Context.txt and ask Copilot to read and understand it.

4

Add Reports

Upload the weekly reports one at a time (or combined). Copilot will include them in the conversation context.

5

Work Incrementally

Since context doesn't persist, upload relevant files at the start of each session. Try the prompts below!
